什么是Quantumult？

Quantumult是一款专为iOS设备开发的代理工具，旨在帮助用户突破地域限制，访问全球互联网资源。通过它，用户可以轻松地绕过网络审查，保护隐私，保证数据的安全性。Quantumult支持多种代理协议，如VPN、HTTP代理等，且具有强大的规则管理、流量监控、订阅管理等功能。

Quantumult的核心功能

  1. 代理功能： Quantumult支持多种代理协议，包括VPN、HTTP代理、Socks5等，用户可以根据需要选择合适的代理方式。

  2. 规则管理： 用户可以自定义规则，指定哪些请求应该使用代理，哪些请求直接通过本地网络进行。

  3. 流量监控： 实时监控网络流量，查看各个应用的带宽占用，帮助用户掌握网络资源的使用情况。

  4. 订阅管理： 允许用户添加、管理各种网络代理订阅，便于快速切换和更新代理配置。

显示订阅有更新功能

Quantumult的“显示订阅有更新”功能主要是为了帮助用户实时了解其所订阅的代理服务是否有新的更新。当你使用Quantumult订阅代理服务时，这一功能能够及时提醒你订阅链接的更新，从而避免因订阅未及时更新导致的连接问题或错误配置。

1.1 如何开启显示订阅有更新功能？

启用这一功能的步骤非常简单，只需按照以下步骤操作：

  1. 打开Quantumult应用。

  2. 进入应用的设置界面。

  3. 在设置界面中找到订阅管理区域。

  4. 勾选“显示订阅有更新”选项。

  5. 确认设置后，功能即可生效。

开启此功能后，每当你所订阅的链接有更新时，Quantumult会通过通知栏或弹窗的方式提醒你，让你第一时间知晓订阅内容的变化。

1.2 显示订阅有更新的工作原理

Quantumult的“显示订阅有更新”功能是通过定期检测订阅链接中的内容是否发生变化来实现的。当订阅链接更新时，Quantumult会推送通知或弹窗，提示用户订阅已更新。用户可以根据提示进行相应的操作，比如手动更新订阅配置或直接忽略更新。

1.3 更新频率设置

Quantumult允许用户自定义订阅更新的频率。一般来说，用户可以设置更新间隔，如每小时更新、每天更新等。根据自己的网络需求和流量限制，用户可以灵活调整更新频率，以避免频繁更新导致不必要的流量消耗。

如何配置Quantumult的订阅？

在Quantumult中配置订阅并开启“显示订阅有更新”功能，用户需要完成一系列的操作。以下是详细的配置步骤：

2.1 添加订阅链接

首先，用户需要准备一个有效的订阅链接。订阅链接通常由提供代理服务的商家或平台提供，用户只需要复制该链接并在Quantumult的订阅设置中进行添加。

添加订阅链接的步骤如下：

  1. 打开Quantumult应用，进入设置界面。

  2. 在设置中找到订阅管理，点击添加订阅。

  3. 将订阅链接粘贴到指定输入框中，点击保存。

通过这一简单的步骤，用户即可完成订阅的添加，开始使用该代理服务。

2.2 修改订阅配置

Quantumult不仅支持添加订阅链接，还允许用户根据需要对订阅配置进行修改。用户可以修改代理服务器的地址、端口、匹配规则等，以满足不同的网络需求。

修改订阅配置的步骤如下：

  1. 进入Quantumult的订阅管理界面。

  2. 选择需要修改的订阅链接，点击编辑。

  3. 根据需要调整订阅设置，修改代理服务器地址、端口等信息。

  4. 确保勾选“显示订阅有更新”选项，便于及时获取订阅内容的更新。

2.3 切换订阅源

有时候，用户可能需要更换或切换到另一个订阅源。这时，只需简单地添加新的订阅链接，删除旧的订阅链接即可。切换订阅源时，Quantumult会自动更新配置，并确保新订阅源的内容能够及时反映到应用中。

常见问题解答（FAQ）

3.1 Quantumult的显示订阅有更新意味着什么？

当Quantumult显示“订阅有更新”时，意味着当前订阅链接中包含有新更新的内容。这通常包括代理节点的变化、配置参数的更新等。用户应根据提示及时更新订阅内容，以确保代理服务的正常运行。

3.2 我能拒绝某个订阅更新提示吗？

是的，Quantumult允许用户手动控制订阅更新提示。如果不希望收到某个订阅的更新提醒，用户可以在设置界面中关闭“显示订阅有更新”功能，或取消某个订阅的更新通知。

3.3 如果订阅更新提示频繁怎么办？

如果发现更新提示频率过高，可能是订阅源频繁变动。此时，用户可以在Quantumult的设置中调整更新的频率，比如将其设置为每天更新一次，或者直接更换更新频繁的订阅源。

3.4 如何删除或更换订阅链接？

如果某个订阅链接不再有效，或者用户想要更换服务提供商的订阅链接，用户可以简单地删除旧的订阅链接并添加新的订阅链接。在订阅管理中找到旧订阅，点击删除，然后按照添加订阅链接的步骤添加新的订阅。

3.5 如何获得最优配置？

为了获得最优配置，用户可以定期检查和更新自己的订阅源。不同的订阅源可能提供不同的代理节点和配置，选择一个稳定且速度较快的订阅源，有助于提升网络体验。

FAQ

Matsuri 如何通过节点测速选择最优线路？
Matsuri 会定期测速节点延迟和丢包率，结合策略组自动选择延迟低、稳定性高的节点。这样可以提升网页、视频和游戏访问速度，减少掉线和卡顿，优化整体体验。
Surfboard 的策略组可以根据延迟自动切换节点吗？
支持。Surfboard 可定期测速节点延迟，并结合策略组自动选择延迟最低的节点。这样用户无需手动切换节点即可保持高速访问，提高网络体验和稳定性。
VMess 配置 TLS+WebSocket 有哪些性能优势？
TLS+WebSocket 可伪装 HTTPS 流量，增强抗封锁能力，同时降低被封检测风险。结合策略组和节点延迟检测，可保证访问速度和连接稳定性，适合视频和游戏场景。
Matsuri 的节点测速结果能影响策略组吗？
可以。Matsuri 会将延迟最低的节点自动标记为优先节点，结合策略组，程序会优先选择最佳节点处理流量，保证访问速度和连接稳定性。
VLESS + XTLS 在移动网络的优势是什么？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定移动网络。结合策略组和节点切换，可提升网页浏览、视频播放和游戏连接的稳定性和流畅度。
Netch 全局与规则代理如何选择？
全局代理将所有流量通过节点转发，适合游戏或全局科学上网；规则代理只针对指定应用、IP 或端口走节点，可优化带宽使用和访问效率，用户可根据需求灵活选择模式。
SSR 混淆方式如何增强隐蔽性？
选择合适混淆方式，如 tls1.2_ticket_auth，可模拟 HTTPS 流量。合理搭配协议和混淆参数，提高节点稳定性和抗封锁能力，确保长期科学上网体验顺畅。
Nekobox 可以定时更新订阅节点吗？
支持。用户可设置定时更新频率，例如每天或每小时，程序会自动下载订阅节点并替换旧节点，确保节点始终可用并保持最新线路信息。
V2Ray 的核心组件有哪些？
V2Ray 的核心由 inbound、outbound、routing、policy 等模块组成。Inbound 处理入站请求，Outbound 负责出站连接，Routing 管理流量分流策略。模块化的设计让用户可以自由组合代理方式。
Sing-box 的性能比 Clash 好吗？
Sing-box 在性能优化上更为出色，特别是在高并发连接下延迟更低。它采用 Go 语言编写并优化了内存占用，比传统 Clash 内核运行更轻盈，尤其适合服务器端部署或移动端设备使用。
